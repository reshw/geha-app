# CrewAI Integration

이 디렉토리는 LoungeAP 프로젝트에 CrewAI를 적용하기 위한 공간입니다.

## 폴더 구조

```
crewai/
├── agents/          # AI 에이전트 정의
├── tasks/           # 태스크 정의
├── tools/           # 커스텀 도구 및 유틸리티
├── crews/           # 크루(에이전트 팀) 정의
├── config/          # 설정 파일 (YAML, JSON 등)
├── main.py          # 메인 실행 파일
└── requirements.txt # Python 패키지 의존성
```

## 설치

```bash
cd crewai
pip install -r requirements.txt
```

## 사용법

```bash
python main.py
```
